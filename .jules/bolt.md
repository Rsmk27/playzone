## 2024-05-25 - Fast Mongoose serialization | **Learning:** Using JSON.parse(JSON.stringify(doc)) to serialize Mongoose documents for Next.js is slow. Replacing it with .lean() and manual stringification of ObjectIds/Dates is much faster. | **Action:** Always use .lean() on Mongoose queries returning data to Next.js Client Components and manually stringify _id and Date fields, avoiding JSON stringify/parse overhead.
