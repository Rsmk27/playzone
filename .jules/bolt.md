## 2026-06-09 - Vitest JSX Configuration | **Learning:** If `vitest` throws errors parsing JSX in files, update `tsconfig.json`'s `jsx` setting to `react-jsx` instead of `preserve`, and use a basic `vitest.config.js` with `plugins: [react()]`. | **Action:** Always configure JSX handling when setting up tests for React components with Vitest in an environment that previously relied solely on Next.js default config for JSX parsing.
